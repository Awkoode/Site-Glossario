Metadados (dentro do Head)

<title>

Serve para mudar o título da página

<meta>

Serve para oferecer informações e metadados como acentos especiais etc.

<link>

Serve para linkar stylesheet e outros tipos de documentos

Títulos e Parágrafos

<h1> até <h6>

Serve para criar títulos

<p>

Serve para criar parágrafos

<br>

Serve para pular uma linha na página

<hr>

Serve para criar uma linha horizontal na página

Formatação de Texto

<b>

Serve para destacar um texto em negrito

<strong>

Mesma coisa do B mas com destaque a mais

<i>

Serve para destacar um texto em itálico

<em>

Mesmo princípio do <i> mas com entonação especial

<u>

Deixa o texto sublinhado

<mark>

Deixa o texto marcado

<small>

Deixa o texto com as letras pequenas

<del>

Mostra o texto riscado

<ins>

Mostra texto sublinhado tambem

<sub>

Deixa o texto Subscrito

<sup>

Deixa o texto Sobrescrito

Listas

<ul>

Cria uma lista não ordenada

<ol>

Cria uma lista ordenada

<li>

Cria um item de destaque com um ponto

<dl>

Cria uma lista de definição

<dd>

Mostra a descrição de um termo

Links e Âncora

<a>

Serve para criar Textos clicáveis e outros links

<button>

Cria um na página que pode levar a outras páginas entre outros.
